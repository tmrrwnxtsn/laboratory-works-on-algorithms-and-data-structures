### Лабораторная работа №6
**Задание**:
1. Реализовать хеш-таблицу, в которой коллизии разрешаются квадратичным пробированием;
2. Реализовать хеш-таблицу, в которой коллизии разрешаются методом цепочек (элемент хеша - связаный список).

Описание исходного кода:
| Файл | Описание |
|----:|:----|
| Item.java | Элемент хеш-таблицы |
| HashTableQuad.java | Хеш-таблица, в которой коллизии разрешаются квадратичным пробированием |
| Node.java | Узел односвязного списка |
| SortedNodeList.java | Односвязный упорядоченный список |
| HashTableChains.java | Хеш-таблица, в которой коллизии разрешаются методом цепочек (элемент хеша - связаный список) |
| HashTablesApp.java | Работа с хеш-таблицами |
