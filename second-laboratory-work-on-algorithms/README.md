**Задание**: реализации дека (двусторонней очереди) на одномерном массиве и двустороннем односвязном списке и сравнение реализаций.

| Файл | Описание |
|----:|:----|
| MainApp.java | Основной класс, в котором происходит работа с деками и сравнение их реализаций |
| Array.java | Реализация дека на одномерном массиве |
| ArrayDeque.java | Работа с деком на одномерном массиве |
| Node.java | Элемент односвязного списка |
| TwoWayNodeList.java | Реализация дека на двустороннем односвязном списке |
| TwoWayNodeListDeque.java | Работа с деком на двустороннем односвязном списке |
